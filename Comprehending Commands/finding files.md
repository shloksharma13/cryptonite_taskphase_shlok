# finding files
In this challenge we learn how to use the command `find`. 
We're given that the flag is stored in a file named `flag` in a random directory. So we use the command `find` with the criteria `-name` and argument `flag`.
This lists all files named flag. Thus we use `cat` command to read out all of these files and finally unlock the flag.
